# express-ts-base

---

## __Develop Environment__
- git
- node
- vs code

### __VS CODE(extension)__
- TSLint

### __NODE package__
- Typescript install  
```bash
npm install --global typescript
```

---

### __Start!!!__
- __Git clone__  
```bash
git clone [ your fork url ]
```

- __Change directory and dependency package install__  
```bash
cd express-ts-base && npm install
```
- __Run server__  
```bash
npm start
```
